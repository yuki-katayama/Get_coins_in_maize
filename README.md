# Get_coins_in_maize
## 日本語
### 概要
迷路の中で、コインを全て獲得するゲーム

### ルールと操作
+ [a, w, s, d]もしくは、[↑, ↓, ←, →]で操作可能
+ 赤い壁に当たるとゲームがリセット
+ コインは接触すると、獲得判定になる

## English
### Summary
A game where you try to get all the coins in the maze.

### Rules and controls
+ Use [a, w, s, d] or [↑, ↓, ←, →] to control the game.
+ The game resets when you hit the red wall.
+ If you touch a coin, you get the coin.
